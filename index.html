TVLinker

tv-release.net link scraper integrated with real-debrid to unrestrict links and aria2, pyLoad or direct download manager options

View project on GitHub

Qt5 based desktop widget for scraping and organising links from various download feeds/sites.

Currently supporting tv-release.net link scraping together with integrating with real-debrid to unrestrict links (a valid real-debrid account and api_token is necessary).

Finally, integration with Aria2 RPC Daemon (windows/linux), pyLoad (linux), IDM (Internet Download Manager; windows only) or built-in direct download to your desktop is all included and easy to configure from the GUI.
